Renishaw inVia Qontor

The Renishaw inVia Qontor is a confocal Raman system built on a Leica microscope and equipped with an automated mapping stage. We recorded spectra of acetonitrile and Chlamydomonas using a 5x and a 50x LWD objective, respectively. We prepared slides of Chlamydomonas and focused using the built-in camera. We acquired the data using the Renishaw WiRE software.

- Acetonitrile Chlamydomonas cells
Wavelength 785 nm 785 nm
Power Not measured 16.3 mW
Exposure 10 s 5–10 s
Num averages 1 5–20

Wasatch WP 785X

The Wasatch WP 785X is a compact system with a probe attachment, integrated illumination source, and no stage. We built a holder and stage for this system. We acquired the data using the Wasatch Photonics ENLIGHTEN software (version 4.1.6).

OpenRAMAN

The OpenRAMAN starter edition is a low-cost, open-source Raman spectroscopy system developed by Luc Boussemaere. We have previously used and described this system in our research (Avasthi et al., 2024; Braverman et al., 2025), but we’ve modified it to achieve higher throughput by adding a motorized stage. We acquired the data using the Spectrum Analyzer Suite (version 1.0.1) and µManager (version 2.0.3).

- Acetonitrile Chlamydomonas cells
Wavelength 532 nm 532 nm
Power 4.5 mW 4.5 mW
Exposure 10 s 5 s
Num averages 5 2
