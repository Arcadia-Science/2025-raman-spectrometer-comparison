Renishaw inVia Qontor

The Renishaw inVia Qontor is a confocal Raman system built on a Leica microscope. It has an automated mapping stage. For this dataset, we used the 5x and 50x LWD objectives for acetonitrile and Chlamydomonas, respectively. We prepared slides of Chlamydomonas and focused using the built-in camera. We acquired the data using the Renishaw WiRE software.

OpenRAMAN

The OpenRAMAN starter edition is a low-cost open-source system, originally conceived by Luc Boussemaere. It uses an external Thorlabs laser, a focusing lens, and a Teledyne CMOS camera. We've modified it to achieve higher throughput via a shutter and a home-built stage to facilitate automation.
